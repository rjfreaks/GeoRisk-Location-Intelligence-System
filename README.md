# 🌍 GeoRisk AI: Disaster Risk Assessment & Location Intelligence Platform

Welcome to the **GeoRisk AI System**! This project leverages machine learning and FEMA National Risk Index (NRI) data to assess disaster risk, social vulnerability, and community resilience across U.S. counties. The goal is to identify safer locations and support data-driven decision-making through risk analytics and visualization.

---

# 📊 Project Overview

In this project, I:

- 📥 Collected and analyzed FEMA National Risk Index (NRI) data.
- 🧹 Performed data cleaning and preprocessing.
- 📈 Conducted Exploratory Data Analysis (EDA) to uncover risk patterns.
- 🧠 Applied K-Means Clustering to group locations based on risk characteristics.
- 📍 Generated Safety Scores using disaster risk, resilience, and vulnerability metrics.
- 📊 Created interactive visualizations for location intelligence.
- ✅ Evaluated clustering performance using Silhouette Score.

---

# 🧠 Machine Learning Approach

The project uses:

- K-Means Clustering
- Feature Engineering
- StandardScaler Normalization
- Safety Score Calculation
- Cluster Analysis
- Silhouette Score Evaluation

### Key Features Used

- Expected Annual Loss (EAL)
- Risk Score
- Social Vulnerability
- Community Resilience
- Natural Hazard Exposure
